# 4u7o-logger

A simple, lightweight, powerful logger module for 4u7o projects.



#### Log Formatter
- **JSON**:
```jsonc
{
    "timestamp": "2024-07-31 14:00:13",
    "level": "INFO", // INFO, WARN, ERROR, TRACE, DEBUG
    "message": "This is the log message sample.",
    "trace": {
        
    }
}
```
